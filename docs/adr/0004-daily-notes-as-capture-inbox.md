# Daily Notes as capture inbox, Notes as durable layer

All Captures default to today's Daily Note as a new Block rather than creating a fresh atomic Note per thought. Atomic Notes are created deliberately — via Cmd+N or by Distilling Blocks from a Daily Note. We rejected "every thought is a Note" (too much ceremony per thought, atomic Notes proliferate as scratch and pollute the knowledge graph), a single growing "Inbox" file (a year of thoughts in one file is unwieldy and the geo dimension loses its temporal anchor), and no-daily-notes-at-all (forces premature crystallization).

This two-tier model matches actual Zettelkasten practice (Luhmann's separate "fleeting notes" pad, then his Zettelkasten proper) and produces the friction asymmetry the workflow needs: Capture is near-zero ceremony, distillation is deliberate. It also gives Geo Stamps a meaningful temporal anchor — a Daily Note becomes a spatial timeline of the day, which is a property no other PKM tool has natively.
